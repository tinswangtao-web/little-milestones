## Context / 用户需求

iPhone 16 Pro / iOS 26.4.1 上使用 Little Milestones 插件体验不佳。用户反馈 4 个阻塞性问题：
1. 打分页编辑项目时，点备注输入框被键盘覆盖，卡片不能上滑
2. 设置页新增打分项后页面跳到"分类管理"顶部，而非留在新增项处
3. 设置页可左右横滑，干扰上下滚动
4. 设置页滚动时误触输入框会弹键盘

## 受影响文件

- `styles.css` — `.kid-score-value-popup` / 新增 `.kid-score-settings-root` / `.kid-score-new-highlight`

## 实现摘要

1. 打分编辑弹窗：popup 加 `max-height`/`overflow-y: auto`；`setupMobileKeyboard` 随 `visualViewport` 压缩 popup 高度；`scrollIntoView` 改 `block:"center"`，延迟 350→60ms；移除移动端自动 focus
2. 设置页新增滚动：`renderCategories` / `renderItems` 保存并恢复 `containerEl.scrollTop`，新项加 `.kid-score-new-highlight` 动画并 `scrollIntoView`
3. 横向滑动锁定：新增 `.kid-score-settings-root` class，`overflow-x: hidden`，子元素 `flex-wrap: wrap; min-width: 0`
4. 滚动误触：`display()` 中给根容器绑 `touchstart/move/end`，位移 >8px 时给全体 input/textarea 加 `readonly`，`touchend` 后 250ms 解除
5. 继续收口 iPhone 键盘与设置页滚动：编辑卡片在键盘弹出时按可视区底边贴齐，不再只是估算上移；手动上下拖动改成不再使用固定 snap 点，允许把卡片拖到能看到最上方或最下方；设置页打分项列表重绘时先保住当前滚动位置，仅在新增项目时再滚到新项并聚焦
6. 数据层安全加固（按用户批准的 ChatGPT 方案执行）：本地日期工具替换 `toISOString().slice(0, 10)` / `new Date(dateStr + "T00:00:00")` 的危险路径；frontmatter 升级为 `schemaVersion: 2` + 结构化 `customItems`；`readDayData()` 兼容旧的字符串数组 `customItems` 和布尔 `scores`；`streak` 与周筛选改用自然日期连续性；目标 callout 读取当前用户 goal；导入配置增加基础结构校验；路径迁移遇到同名文件时停止并报错，不再静默覆盖
